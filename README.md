# Task Board (Mini-Trello)

A drag-and-drop Kanban board for task management with local persistence.

## Features

- **Three Columns**: Todo, In Progress, Done
- **Drag & Drop**: Move tasks between columns and reorder within columns
- **Task Management**: Create, read, update, and delete tasks
- **Task Metadata**: Priority, tags, due dates
- **Search & Filter**: Global search by title/description/tags, priority filter
- **Local Storage**: All data persists in browser local storage

## Deployment

This application can be deployed to Vercel:

1. Push the code to a GitHub repository
2. Connect the repository to Vercel
3. Deploy with zero configuration

## Local Development

1. Clone or download the project files
2. Open `index.html` in a browser
3. No build process or dependencies required

## Usage

- Click "Add Task" to create a new task
- Drag and drop tasks between columns to update their status
- Click on any task to edit its details
- Use the search box to filter tasks by content
- Use the priority dropdown to filter by priority level
